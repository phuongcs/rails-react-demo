# README

Rails 7 webpacker with Reactjs on Frontend

## System

- Ruby version v3.1.3
- Rails v7.0.4

## Installation

```
bundle
yarn
```

## Run

```
rails s
```

## Important Gems
### react-rails
for rendering React component with prop from controller

generate react component
```
rails g react:component home
```
this will generate Home component.
